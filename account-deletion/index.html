계정 삭제 안내

Claspeek 계정 삭제 절차와 삭제되는 데이터에 대해 안내합니다.

1. 계정 삭제 방법

Claspeek 앱 내에서 직접 계정을 삭제할 수 있습니다.

  1. Claspeek 앱을 실행하고 로그인합니다.
  2. 마이페이지로 이동합니다.
  3. 설정을 선택합니다.
  4. 개인정보 및 보안을 선택합니다.
  5. 계정 삭제를 선택합니다.
  6. 안내 내용을 확인한 뒤 삭제 버튼을 누르면 계정이 삭제됩니다.
주의: 계정을 삭제하면 모든 데이터가 영구적으로 삭제되며, 복구할 수 없습니다.

2. 삭제되는 데이터

계정 삭제 시 다음 데이터가 삭제됩니다.

데이터 유형상세삭제 시점
계정 정보닉네임, 이메일, 로그인 ID, 소셜 로그인 연동 정보즉시
프로필 정보프로필 이미지, 학교/학과 정보, 학년즉시
사용자 생성 콘텐츠게시글, 댓글, 투표, 다짐, 일기, 일정, 할 일즉시
활동 기록좋아요, 북마크, 조회 기록즉시
기기 정보기기 식별자, 푸시 알림 토큰즉시
학생 인증 정보학생증 인증 기록즉시

3. 법령에 따른 보관 데이터

관련 법령에 의해 일부 데이터는 계정 삭제 후에도 일정 기간 보관될 수 있습니다.

보관 항목보관 기간근거 법령
계약 또는 청약 철회에 관한 기록5년전자상거래법
소비자 불만 또는 분쟁 처리에 관한 기록3년전자상거래법
로그인 기록3개월통신비밀보호법

보관 기간이 경과하면 해당 데이터는 지체 없이 파기합니다.

4. 앱에 접근할 수 없는 경우

앱에 로그인할 수 없거나 기타 사유로 직접 삭제가 어려운 경우, 아래 이메일로 계정 삭제를 요청하실 수 있습니다.
